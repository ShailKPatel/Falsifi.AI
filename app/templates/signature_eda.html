Signature Verification - EDA & Model Selection

Comprehensive analysis of our Siamese Neural Network for signature verification, including dataset exploration, contrastive learning methodology, and performance evaluation.

Siamese Network Architecture

Network Design

  • Input: Two grayscale signature images (200×320)
  • Backbone: Shared encoder with flatten layer
  • Architecture: Linear(200*320 → 128)
  • Output: 128-dimensional embeddings
  • Similarity: Euclidean distance measure

Training Configuration

  • Loss Function: Contrastive Loss (margin=1.0)
  • Optimizer: Adam (lr=1e-4)
  • Epochs: 20
  • Batch Size: 128
  • Device: CUDA/CPU support

Contrastive Loss Function

L = (1 - y) · D² + y · max(0, m - D)²

Parameters

  • y ∈ {0,1}: Label (0 = similar, 1 = dissimilar)
  • D: Euclidean distance between embeddings
  • m = 1.0: Margin parameter

Objectives

  • Similar pairs (y=0): Embeddings close together
  • Dissimilar pairs (y=1): Embeddings farther than margin
  • Optimization: Minimize contrastive loss

Implementation Details

Model Evaluation Code

import os
import torch
import torch.nn.functional as F
from torch.utils.data import DataLoader, Dataset
from torchvision import transforms
from PIL import Image
import pandas as pd
import numpy as np
from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score
from tqdm import tqdm

# --- Dataset class ---
class SignaturePairDataset(Dataset):
    def __init__(self, csv_file, transform=None):
        self.pairs_df = pd.read_csv(csv_file)
        self.transform = transform
        self.base_path = '/content/'  # Adjust if needed

    def __len__(self):
        return len(self.pairs_df)

    def __getitem__(self, idx):
        row = self.pairs_df.iloc[idx]
        img1_path = os.path.join(self.base_path, row['path1'].replace('\\', '/'))
        img2_path = os.path.join(self.base_path, row['path2'].replace('\\', '/'))
        label = torch.tensor(row['label'], dtype=torch.float32)
        img1 = Image.open(img1_path).convert('L')
        img2 = Image.open(img2_path).convert('L')
        if self.transform:
            img1 = self.transform(img1)
            img2 = self.transform(img2)
        return img1, img2, label

# --- Model definition ---
class SiameseNetwork(torch.nn.Module):
    def __init__(self):
        super(SiameseNetwork, self).__init__()
        self.flatten = torch.nn.Flatten()
        self.fc = torch.nn.Linear(200 * 320, 128)

    def forward_once(self, x):
        return self.fc(self.flatten(x))

    def forward(self, input1, input2):
        return self.forward_once(input1), self.forward_once(input2)

# --- Setup ---
test_csv = "test.csv"
model_weights_path = "Siamese/siamese_model_weights.pt"

device = torch.device("cuda" if torch.cuda.is_available() else "cpu")

# --- Transforms ---
transform = transforms.Compose([
    transforms.Resize((200, 320)),
    transforms.ToTensor(),
    transforms.Normalize((0.5,), (0.5,))
])

# --- Load test data ---
test_dataset = SignaturePairDataset(test_csv, transform)
test_loader = DataLoader(test_dataset, batch_size=128, shuffle=False, num_workers=4)

# --- Load model weights ---
model = SiameseNetwork().to(device)
model.load_state_dict(torch.load(model_weights_path, map_location=device))
model.eval()

# --- Evaluation ---
all_labels = []
all_distances = []

with torch.no_grad():
    for img1, img2, label in tqdm(test_loader, desc="Testing"):
        img1, img2 = img1.to(device), img2.to(device)
        output1, output2 = model(img1, img2)
        distances = F.pairwise_distance(output1, output2)
        all_distances.extend(distances.cpu().numpy())
        all_labels.extend(label.cpu().numpy())

# --- Threshold tuning ---
all_distances = np.array(all_distances)
all_labels = np.array(all_labels)

best_thresh = 0
best_acc = 0
for t in np.linspace(0.1, 2.0, 100):
    preds = (all_distances < t).astype(int)
    acc = (preds == all_labels).mean()
    if acc > best_acc:
        best_acc = acc
        best_thresh = t

# --- Final predictions at best threshold ---
final_preds = (all_distances < best_thresh).astype(int)

# --- Metrics ---
acc = accuracy_score(all_labels, final_preds)
prec = precision_score(all_labels, final_preds)
rec = recall_score(all_labels, final_preds)
f1 = f1_score(all_labels, final_preds)

print(f"\nEvaluation on Test Set:")
print(f"Best Threshold: {best_thresh:.3f}")
print(f"Accuracy     : {acc:.4f}")
print(f"Precision    : {prec:.4f}")
print(f"Recall       : {rec:.4f}")
print(f"F1 Score     : {f1:.4f}")
